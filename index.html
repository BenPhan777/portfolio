Beige

Built with: html, css

Beige is a website for a local clothing store. It is one of my first projects and; hence, I designed it using only HTML and CSS. To complete this project, I've incorporated and strengthened my knowledge of new materials such as building a navigation bar, organize and centering items with grid and flexbox. I also had fun making the website more interactive with CSS's ":hover" and scaling; and giving it a more atheistically pleasing look with new designing approaches.

PineAble

Built with: html, css

A fill-in form for a mystery fruit and vegetable basket service called PineAble made from HTML and CSS. I created this website mainly to practice with the different types of user input tags such as input-radio, text area, datalist, etc. During the project, I faced a few problems regarding margin and labeling tags, which took me quite a bit of time to understand and use correctly. However, I also learned other useful CSS features such as "inline-block" and "transform" that I can use for my other projects.

Dolly Parton

Built with: html, css

The website is a tribute page for the one and only Dolly Parton. It is also my first web project where I applied my basic HTML and CSS to create a both functional and aesthetically pleasing website. Tic Tac Toe

Built with: java

Tic Tac Toe was my first java project. Through this project, I've acquired a deeper understanding of a few different classes in Java, such as Array (mostly char array) and Scanner (processing user input). The most challenging part of this project was to determine the result of the board game; For example, I had to loop through the board in every direction to find the winning pattern, and there was a lot of "out of range" errors. Moreover, I also tried to improve my testing skill by doing a throughout unit-testing for each method whenever possible. And, making my code more readable with appropriate documentation to form a habit for later projects. Overall, Tic Tac Toe was not an easy first project (at least for me); however, I came out of it with a better appreciation for the Java language and gained a few knowledge along the way.
